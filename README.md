# Focus Tchad VSCode 🚀

**Focus Tchad** est une extension conçue pour aider les développeurs à rester concentrés et à sécuriser leur travail. Elle surveille votre activité et vous envoie des rappels de motivation si vous oubliez d'enregistrer vos fichiers.

## Fonctionnalités
- ⏱️ **Tracking intelligent** : Détecte l'inactivité de sauvegarde après modification.
- 💡 **Motivation** : Affiche des citations inspirantes pour vous encourager.
- ⚙️ **Configurable** : Choisissez votre propre délai de rappel dans les paramètres.

## Configuration
Vous pouvez modifier le délai de rappel (en minutes) dans les paramètres de VS Code :
`Focus Tchad > Delai Rappel`
## Installation 🛠️
Téléchargez la dernière version `.vsix` depuis l'onglet [Releases]([lien-vers-ta-release](https://github.com/TomkerDev/focus-tchad-vscode/releases)) et installez-la dans VS Code.
